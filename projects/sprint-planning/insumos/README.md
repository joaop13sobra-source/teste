# Insumos — Sprint Planning

Coloque aqui arquivos de contexto para enriquecer o Sprint Planning de cada projeto.

## O que colocar aqui

| Tipo | Exemplo |
|---|---|
| Prints de métricas | screenshot do Meta Ads, Google Ads, GA4 |
| Relatórios anteriores | sprint planning da semana passada |
| ATAs de reunião | ata do check-in da semana |
| Feedbacks do cliente | prints de conversa, e-mails |
| Planilhas de resultados | export de leads, CPL por campanha |

## Como usar

Ao invocar `/sprint-planning`, informe se há arquivos aqui relevantes:

> `/sprint-planning Techmax — tem ata do check-in em insumos/techmax-ata-08-05.md`

A skill vai ler o arquivo e incorporar no contexto antes de montar o documento.
